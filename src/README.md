# Auto Body Shop Website

This project is a multi-page static website for an auto body shop. It includes various pages that provide information about the shop, its services, and how to contact them.

## Project Structure

The project has the following structure:

```
chava_example
├── index.html          # Home page with a hero section and features
├── about.html          # Introduction to the auto body shop
├── gallery.html        # Gallery of the shop's work
├── contact.html        # Contact information and hours of operation
├── css
│   └── style.css       # Shared styles for the website
└── services
    ├── paint-and-cosmetic
    │   ├── auto-painting.html      # Description of auto painting service
    │   ├── car-detailing.html       # Description of car detailing service
    │   ├── custom-paint.html        # Description of custom paint service
    │   ├── dent-removal.html        # Description of dent removal service
    │   ├── paintless-dent-removal.html # Description of paintless dent removal service
    │   └── rust-removal.html        # Description of rust removal service
    └── collision-and-frame
        ├── bumper-repair.html       # Description of bumper repair service
        ├── collision-repair.html     # Description of collision repair service
        ├── frame-alignment.html      # Description of frame alignment service
        └── frame-straightening.html  # Description of frame straightening service
```

## Pages Overview

- **Home Page (index.html)**: Features a hero section with a background image, a headline, a subhead, and a call-to-action button with the shop's phone number.
  
- **About Page (about.html)**: Provides an introduction to the shop, including the year established, a list of services, payment options, and languages spoken.
  
- **Gallery Page (gallery.html)**: Displays images of the shop's work using stock placeholder images.
  
- **Contact Page (contact.html)**: Contains the shop's address, phone numbers, hours of operation, and holiday closures.

## Styles and Scripts

- **CSS (css/style.css)**: Contains shared styles for the website, including styles for the sticky header and footer, layout, and color scheme.
  
- **JavaScript (js/main.js)**: This file is intentionally left blank for future enhancements.

## Assets

- **Assets Folder (assets)**: Contains any additional assets needed for the website. Currently, it includes a `.gitkeep` file to maintain the folder structure.
