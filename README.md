# Moises Auto Body & Paint

Welcome to the Moises Auto Body & Paint project! This is a static site built with Astro, designed to showcase our auto body and paint services.

## Project Structure

The project is organized as follows:

```
moises-auto-body-paint
├── src
│   ├── components          # Reusable components for the site
│   ├── layouts             # Layout files for consistent styling
│   ├── pages               # Individual pages of the site
│   └── styles              # Global styles for the site
├── public                  # Public assets like images and favicons
├── astro.config.mjs       # Configuration settings for Astro
├── package.json            # Project dependencies and scripts
├── tsconfig.json          # TypeScript configuration
└── README.md               # Project documentation
```

## Getting Started

To get started with the Moises Auto Body & Paint project, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd moises-auto-body-paint
   ```

2. **Install Dependencies**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the Development Server**
   Start the local development server with:
   ```bash
   npm run dev
   ```

4. **Build the Project**
   To build the project for production, run:
   ```bash
   npm run build
   ```
